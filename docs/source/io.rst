IO API
=====================
ZeusAI.py's IO API is used for interacting with the ZeusAI server I/O API.
It can be found in the zeusai\_py.io subpackage.

Running the package directly will launch an interactive CLI client, which is
the simplest way to interact with the ZeusAI server.
